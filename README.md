💰 Expense Tracker

📌 Project Overview:-
Expense Tracker is a Python-based application developed as part of the DecodeLabs Python Programming Internship. The project helps users record, manage, and analyze their daily expenses efficiently.

🚀 Features:-
>Add New Expenses
>View All Expenses
>Search Expenses
>Delete Expenses
>Calculate Total Spending
>Calculate Average Spending
>Display Highest Expense
>Categorize Expenses (Food, Travel, Shopping, etc.)

🛠 Technologies Used:-
>Python
>Lists
>Dictionaries
>Functions
>Loops
>Conditional Statements
